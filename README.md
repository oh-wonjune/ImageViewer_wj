## typescript-imageviewer



설치 방법

    npm install typescript-imageviewer




사용 방법

    import {ImageViewer} from "typescript-imageviewer";
    
    
    ....
    
    <ImageViewer
      url={urlarr}
    />
  
  
  
    url 타입  : list 형식
